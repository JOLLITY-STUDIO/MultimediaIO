# MultimediaIO

🎨 在线图片转 SVG 转换工具

## 功能特性

- 📷 支持 PNG、JPG、JPEG、GIF、BMP 等图片格式
- 🎯 可调节颜色阈值
- ✨ 可调节模糊半径
- 📐 可调节简化容差
- 💾 一键下载 SVG 文件
- 📱 响应式设计，支持移动端

## 使用方法

1. 访问 https://JOLLITY-STUDIO.github.io/MultimediaIO/
2. 点击或拖拽图片到上传区域
3. 调整参数（可选）
4. 点击"转换为 SVG"按钮
5. 预览并下载 SVG 文件

## 本地运行

直接用浏览器打开 `index.html` 即可使用。

## 技术栈

- 纯 HTML/CSS/JavaScript
- Canvas API 图像处理
- 无外部依赖
